## [1.0.0] - 2025-01-15

### Added
- Initial release of ODIN Market Feed Client
- WebSocket client for real-time market data streaming
- Support for Touchline subscription
- Support for Best Five subscription
- Support for LTP Touchline subscription
- ZLIB compression support for data transmission
- Automatic message fragmentation and defragmentation
- Pause/Resume functionality for subscription management
- Event-driven architecture with onOpen, onMessage, onError, onClose handlers
- Full TypeScript support with type definitions
- Comprehensive documentation and examples

### Features
- Real-time market data streaming
- Multiple subscription types
- Bandwidth management with pause/resume
- Type-safe API with TypeScript
- Automatic reconnection handling
- Error handling and logging
